# StarCharge for China

## 项目简介

为中国大陆地区的星星充电（StarCharge）个人充电桩提供 Home Assistant 数据接入。

## 功能

* 在 Home Assistant 中以传感器 (Sensors) 实时显示充电桩的详细数据：
    * 电压 (Voltage)
    * 电流 (Current)
    * 功率 (Power)
    * 充电状态 (Status)
